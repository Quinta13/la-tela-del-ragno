# 🕷️ La Tela del Ragno

Sito realizzato per il **Campetto di Specialità "Esperto del Computer" AGESCI**.

## Scopo del progetto

Questo repository ospita il sito web del campetto e i lavori realizzati dagli **Esploratori e dalle Guide** partecipanti.

Durante il campo, i ragazzi imparano le basi dello sviluppo web attraverso tecnologie semplici e accessibili:

* HTML
* CSS
* Bootstrap
* Pubblicazione di siti statici

L'obiettivo non è soltanto imparare a scrivere codice, ma soprattutto comprendere come funziona il web e utilizzare gli strumenti digitali per raccontare esperienze, idee e attività in modo creativo e responsabile.

## Tema

Il sito principale è ambientato nella redazione immaginaria della **"Tela del Ragno"**, un giornale che si propone di verificare le informazioni e combattere le fake news che circolano su Spider-Man.

Da questa pagina principale è possibile accedere ai siti realizzati dai partecipanti, ciascuno dedicato al racconto della propria esperienza e delle attività svolte durante il campetto.

## Struttura del repository

```text
.
├── index.html
├── style.css
├── site1/
├── site2/
├── site3/
└── site4/
```

Ogni cartella `siteX` contiene il sito sviluppato da un gruppo o da un partecipante.

## Tecnologie utilizzate

* HTML5
* CSS3
* Bootstrap 5
* GitHub Pages

## Pubblicazione

Il sito è pubblicato tramite GitHub Pages e può essere consultato direttamente dal browser senza necessità di installare software aggiuntivo.

## AGESCI

Questo progetto è stato realizzato nell'ambito delle attività della branca **E/G (Esploratori e Guide)** dell'AGESCI, con l'obiettivo di sviluppare competenze digitali, creatività, collaborazione e spirito di scoperta attraverso il metodo scout.

Buona navigazione e... buona caccia alle fake news! 🕸️
